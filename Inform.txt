Локальный запуск проекта:
1. Скачать zip файл и распоковать.
2. Открыть проект.
3. Установить и привязять зависимости, набрав в терминале команду npm i и дождаться установки всех зависимостей.
4. В файле package.json запустить скрипт "start" или в терминале набрать npm run start

Данные для входа содержатся в папке public/users.json
Продублированные данные:

    "username": "user",
    "password": "123"

    "username": "admin",
    "password": "123"

    "username": "person",
    "password": "123"

На выполнение данного задания было потрачено 3 дня работая по 3-4 часа
